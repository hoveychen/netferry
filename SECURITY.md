# Security Policy

Thanks for helping keep NetFerry secure.

## Supported Versions

Only the latest commit on the default branch (`main`) is currently supported for security fixes.

## Reporting a Vulnerability

Please do not disclose vulnerability details in public issues.  
Report security concerns privately using:

- GitHub: [hoveychen](https://github.com/hoveychen)

Please include:

- Scope and impact
- Reproduction steps
- Exploitability conditions
- Suggested mitigation (if available)

We will review and respond as quickly as possible.
